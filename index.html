Keep Notes

This is a simple website but has some exciting features. For example, users can create notes and then search by note title. The website also has some validation: all note fields must be filled to be able to submit, the note title input field has length validation, and notes can't have the same title. Users can switch from Light Mode to Dark Mode. Notes and chosen Mode get saved in localStorage, so users don't lose data by closing the browser. And, of course, this website is fully responsive.
